## Javascript Challenge: Making a Point of Sales Web App
Tools used:
1. Javascript
2. Node.js
3. PostgreSQL
4. Bootstrap
5. Express
6. SB Admin 2 Template

## Features
1. Dashboard
2. Report Generator
3. Goods Management
4. Supplier Management
5. Customer Management
6. User Management
7. Purchase Transaction
8. Sales Transaction
9. Notification
